Сервис поиска и инвентаризации небезопасных криптографических сертификатов.

Сервис может по списку IPv4 адресов получить информацию о сертификатах ресурсов. Главные достоинства:

  • Гибкая настройка параметров безопасности сертификатов
  • Весь функционал доступен через сайт
  • Предоставление результатов анализа
  • Сохранение результатов в базе данных для повторного использования
